```
用法: freqtrade install-ui [-h] [--erase] [--prerelease]
                            [--ui-version UI_VERSION]

选项:
  -h, --help            显示帮助信息并退出
  --erase               清空 UI 文件夹，不下载新版本。
  --prerelease          安装最新的 FreqUI 预发布版本。
                        不建议在生产环境中使用。
  --ui-version UI_VERSION
                        指定要安装的 FreqUI 具体版本。
                        不指定则安装最新版本。
```
